# curso-swiftui

O presente projeto feito no curso Foundation IOS do Open Academy Mackenzie teve sua construção feita das 13:00 as 15:00 durante 10 dias, sendo os 5 primeiros dias focados apenas em aulas teóricas sobre o ambiente Xcode e a linguagem SwiftUI. Decidimos desenvolver um aplicativo que simula uma corretora de ações de ativos Norte-Americanos com a API TwelveData e tambem uma "casa de câmbio" para compra de moedas estrangeiras. O aplicativo em si, PeciMoney, não está completamente pronto e funcional, mas o mesmo, até o presente momento, possui uma tab bar com duas opções, Tela Inicial e Perfil. Perfil mostra suas informações pessoais de usuário, saldo e dados para transação bancaria. Ja na tela inicial temos mais uma vez o saldo e dois botões adicionais, Investir e Consultar. Consultar te permite buscar um ativo para conseguir informações como o preço atual (em Dólar) desse respectivo ativo, já investir te permite comprar ações do ativo da sua escolha e, posteriormente, depois da API carregar com o novo preço do ativo, você pode vende-las caso veja vantagem, adquirindo lucro.
